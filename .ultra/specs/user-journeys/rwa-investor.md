## 一、用户画像

### 1.1 典型用户特征

**用户类型**:
- **小额投资者**: 门槛低（$100 起），无法进入传统 RWA 市场（$100K+）
- **被动收益追求者**: 寻求稳定、可预期的固定收益（5-8% APR）
- **链上原生用户**: 熟悉 DeFi，但希望降低加密货币波动风险
- **跨境投资者**: 传统渠道无法投资美国国债/债券的国际用户
- **资产配置者**: 希望分散投资组合，增加 RWA 敞口

**资金规模**:
- 入门级: $100-$1,000
- 中等级: $1,000-$10,000
- 大户: $10,000-$100,000
- 巨鲸: $100,000+

**技术能力**:
- 基础: 会使用 MetaMask 钱包、理解 USDC
- 中级: 了解 KYC、白名单机制
- 高级: 理解 NAV、到期赎回、清算机制

### 1.2 用户价值主张

| 传统 RWA 投资 | Paimon.dex Launchpad | 优势 |
|--------------|---------------------|------|
| 最低投资 $100K | 最低投资 $100 | **门槛 ↓99.9%** |
| 锁定期 3-12 个月 | 二级市场 T+0 退出 | **流动性 ↑100%** |
| T+30 结算 | T+0 结算 | **效率 ↑100%** |
| 中介费 3-5% | 平台费 0.5-1% | **成本 ↓80%** |
| 仅合格投资者 | 全球开放（KYC） | **覆盖 ↑1000%** |

**核心卖点**:
- ✅ **低门槛**: $100 起投，vs 传统 $100K
- ✅ **高流动性**: DEX 二级市场 24/7 交易，即时退出
- ✅ **透明度**: 链上实时查询 NAV、赎回价格、抵押率
- ✅ **固定收益**: 美国国债 5.25% APR、房地产租金 6-8% APR
- ✅ **全球可及**: 无地域限制（符合 KYC 的地区）

#### 步骤 1.3: 完成 KYC（首次需要）

**KYC 流程**（Blockpass 集成）:
```
步骤 1: 点击 [开始 KYC] → 跳转 Blockpass 页面

步骤 2: 填写基本信息
  - 全名（First & Last Name）
  - 出生日期（Date of Birth）
  - 国籍（Nationality）
  - 居住地址（Residential Address）

步骤 3: 上传身份证件
  - 护照 / 驾照 / 身份证
  - 拍照上传（正反面）

步骤 4: 人脸识别（Liveness Check）
  - 自拍（检测是否真人）
  - 与证件照比对

步骤 5: 提交审核
  - 审核时间: 1-24 小时（自动 + 人工）
  - 通过后返回 Paimon.dex

⚠️ 隐私保护:
  - KYC 数据加密存储，仅 Blockpass 和监管机构可访问
  - Paimon.dex 仅知道你的地址是否通过 KYC（是/否）
  - 不存储任何个人信息在链上
```

## 四、常见问题（FAQ）

### Q1: KYC 必须完成吗？

**A**: 取决于项目。
```
必须 KYC:
  - 美国国债类（SEC 法规要求）
  - 房地产类（AML 合规）
  - 发行方所在国法律要求

无需 KYC:
  - 部分 T3 资产（发行方自主决定）
  - 二级市场交易（DEX 购买 pUST125）

KYC 一次，全平台通用:
  完成 KYC 后，参与所有需要 KYC 的项目无需重复
```

### Q2: 如果发行方跑路怎么办？

**A**: 多重保护机制。
```
1. 托管机制:
   发行方无法直接提取投资者资金
   资金存入 IssuanceController 智能合约
   只能通过合约逻辑分配

2. 抵押品验证:
   第三方托管机构（如 State Street）持有国债
   Chainlink Proof of Reserve 每日验证
   NAV 与 Chainlink 价格偏差 >20% → 暂停铸币

3. 清算机制:
   如果 NAV 持续 <$0.98 超过 14 天
   DAO 可投票启动紧急清算
   优先保护投资者本金

4. 保险（可选）:
   部分项目购买 Nexus Mutual 智能合约保险
   覆盖合约漏洞风险
```

```
合约层面:
  ❌ 大部分项目不支持提前赎回（流动性风险）
  ✅ 少数项目支持（需支付 3-5% 提前赎回费）

DEX 出售:
  ✅ 任何时候都可在二级市场出售 pUST125
  价格: 由市场决定（可能溢价或折价）
  流动性: 取决于 DEX 池子深度（通常 $500K-$2M）

建议:
  如果急需流动性 → DEX 出售（承受 0-2% 价差）
  如果不急 → 持有至到期（获得完整收益）
```

### Q4: NAV 为什么会波动？

**A**: 多种因素影响。
```
影响因素:
1. 利率变化:
   美联储加息 → 国债价格下跌 → NAV ↓
   示例: 6 个月国债收益率 5.25% → 6.00%
         NAV 可能从 $1.00 跌至 $0.97

2. 信用风险:
   公司债评级下调 → NAV ↓
   房地产空置率上升 → 租金收入 ↓ → NAV ↓

3. 市场流动性:
   恐慌抛售 → NAV 短期偏离真实价值

4. 汇率波动（如果有非美元资产）:
   例如欧洲房地产，欧元贬值 → NAV ↓

应对策略:
  - 选择 T1 资产（波动最小）
  - 长期持有（短期波动不影响到期收益）
  - 分散投资（不同资产类型）
```

### Q5: 二级市场价格为什么不是 $1.00？

**A**: 市场供需决定。
```
价格 > $1.00（溢价）:
  原因:
    - 需求 > 供应（投资者抢购）
    - 剩余收益率高（接近分红日）
    - NAV 上涨预期

  示例: 分红前 1 周
    pUST125 DEX 价格: $1.015
    原因: 买家买入后可立即领取 $13.13 分红

价格 < $1.00（折价）:
  原因:
    - 供应 > 需求（持有者恐慌抛售）
    - NAV 下跌（利率上升）
    - 流动性不足（买盘枯竭）

  示例: 美联储突然加息
    pUST125 DEX 价格: $0.995
    原因: 市场预期 NAV 下跌，提前折价出售

套利机会:
  如果价格 < NAV → 买入 → 到期赎回
  如果价格 > (NAV + 剩余收益) → 卖出 → 买入国债 ETF
```

## 五、风险提示

### 5.1 市场风险

⚠️ **利率风险**
- 美联储加息 → 国债价格 ↓ → NAV < $1.00
- 缓解: 选择短期资产（3-6 个月）

### 5.2 流动性风险

⚠️ **DEX 流动性不足**
- 如果需提前退出，可能需折价 2-5%
- 缓解: 选择 TVL >$1M 的项目

### 5.3 信用风险

⚠️ **发行方违约**
- 公司债违约 → NAV 大幅下跌
- 房地产空置 → 租金收入中断
- 缓解: 选择 T1/T2 资产（投资级）

### 5.4 智能合约风险

⚠️ **合约漏洞**
- IssuanceController 漏洞 → 资金被盗
- 缓解: 审计报告（CertiK）+ 保险（可选）

### 5.5 监管风险

⚠️ **政策变化**
- 稳定币新法规 → 项目暂停
- RWA 证券法监管 → 强制 KYC/AML
- 缓解: 关注监管动态，分散投资
