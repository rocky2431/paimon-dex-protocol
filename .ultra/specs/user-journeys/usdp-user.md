
**获取 USDC**:
```
方式 1: 中心化交易所提现
  币安 → BSC 网络提现 USDC → 你的 MetaMask 地址

```

```
推荐准备: 0.05 BNB（~$20，够用 50-100 次交易）

- 币安提现 BNB 到 BSC 地址
```

#### 步骤 2.3: 执行存款

**交易 2: Deposit to SavingRate**

**涉及合约**: `USDPSavingRate` (0x...)

```solidity
// USDPSavingRate.sol (基于 ERC4626)
function deposit(
    uint256 assets,    // 1000e18 USDP
    address receiver
) external returns (uint256 shares);
```

**执行细节**:
```
输入: 1,000 USDP
输出: ~1,000 sUSDP（Share Token，会计代币）

汇率计算:
shares = assets * totalShares / totalAssets
      = 1,000 * 10,000,000 / 10,050,000  // 假设池子已有收益
      = 995.024 sUSDP

Gas 费: ~0.0008 BNB (~$0.32)
```

**sUSDP 机制说明**:
```
sUSDP 是 Share-based 会计代币，代表你在池子中的份额。

示例:
时间 T0（存款时）:
  - 池子总资产: 10,050,000 USDP（含累积收益）
  - 池子总份额: 10,000,000 sUSDP
  - 汇率: 1 sUSDP = 1.005 USDP
  - 你存入 1,000 USDP → 获得 995.024 sUSDP

时间 T1（30 天后）:
  - 池子总资产: 10,270,625 USDP（收益继续累积）
  - 池子总份额: 10,000,000 sUSDP（不变）
  - 汇率: 1 sUSDP = 1.027 USDP
  - 你的 995.024 sUSDP → 价值 1,021.92 USDP（收益 $21.92）
```

#### 步骤 2.4: 收益复利累积

**自动复利**:
```
SavingRate 采用 ERC4626 标准，收益自动复利：

每周协议费分配（EmissionRouter 触发）:
  function distributeProtocolFee(uint256 usdpAmount) external {
      // 直接增加池子总资产
      totalAssets += usdpAmount;
      // sUSDP 总量不变，单价上涨 = 复利
  }

用户无需任何操作，sUSDP 自动增值。
```

**收益来源明细**:
| 来源 | 占比 | 年化收益 | 说明 |
|------|------|---------|------|
| **USDP 协议费** | 50% | 1.2% | DEX 交易手续费、Treasury 铸造费 |
| **RWA 利息分成** | 20% | 0.5% | RWA 资产收益的 10% 分给储蓄池 |

### Q1: USDP 的 1:1 锚定如何保证？

**A**: 三重机制保障。
```
机制 1: PSM 套利机制
```
